All the files that are required to use BlazeGears can be found in the “Modules” directory.
The BlazeGears class declaration interface is largely based on the concepts used by the Python language.
The HTML templating class uses the BlazeGears Templating Language, which is based on a mix of JavaScript and BottlePy’s Simple Template.
FragVars are variables stored in the fragment section of the current URL.
The Form class puts HTML forms into an object-oriented context.
The Paginator class is technically a cooperation between a managed FragVar and a BGTL template.
XML request are done by a wrapper around either the XMLHttpRequest object or an ActiveX component, depending on the browser.
